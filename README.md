# Text_Adventure_The_Cedwick_Curse
A simple text adventure to help me apply the things I've learned in my beginning Python class. A mash-up of my newly learned coding skills and my enjoyment of interactive stories.
Consists of an original branching story written by me(Matt Dinoff)

Players will be taken on adventure to attempt to save a little ghost girl named Aelita from her tormentor, the Shade.
Players will make choices that effect the story.
